Seasons Home Page

On this website you can click on the different seasons in the navigation, where each tab will take you to a different page telling you about the season you have selected. Along with basic information about the season the page will also contain a picture representing that season. I have created this website for an assignment in my web design class at Carroll College. Although I am no expert on the seasons I believe I can create a visually appealing website about them!
